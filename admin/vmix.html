URL vMix Alertes

Génère l'URL signée pour vMix Browser Input.

Générer l'URL

L'event_id se trouve dans l'URL de la régie : /admin/live.html?id=UUID-ICI

URL vMix prête

Dans vMix : Add Input → Web Browser → coller l'URL → OK. Taille recommandée : 480×270 ou 520×360.

Mode d'emploi

  1. Récupère l'event_id dans l'URL de la régie de l'event en cours
  2. Colle-le ci-dessus et clique Générer
  3. Clique Copier l'URL
  4. Dans vMix : Add Input → Web Browser → colle l'URL → valider
  5. Place ce Browser Input en overlay sur ton multiview ou en PIP
  6. Au moment où un participant signale un problème, l'alerte apparaît dans vMix en ~2 sec
