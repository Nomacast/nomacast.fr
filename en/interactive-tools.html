Frequently asked questions

About interactive tools.

The most common questions on configuration, integration, and operation during the event.

Any live broadcast with an identifiable audience: conferences, keynotes, seminars, general meetings, conventions, webinars, panel discussions. The solution integrates particularly well with existing Nomacast setups (multi-camera 4K production, corporate live shows, multi-site multiplex).
Three simple steps: (1) configuration on this page, custom quote within 24 hours with a unique configuration ID; (2) upon quote approval, you provide the participant list as a CSV and your finalized branding guidelines; (3) invitations are sent by email 5 to 7 days before the event. No software installation required on the participant side.
Each participant receives via email a secure named access link that authenticates them automatically in one click, with no account creation or password to remember. You provide an invitee list as a CSV (email, name, company) and invitations are sent before the event. A login + password access mode is also available if your context requires it.
The named access link covers the vast majority of one-off event needs. User login access is relevant in three cases: recurring event with the same participants, integration with your corporate directory, or requirement for an event password. Specify your need in your quote request so we can study the most suitable configuration.
Yes. You receive a ready-to-use iframe embed code for any web page (corporate site, intranet, event landing page). The interface automatically adapts to the available width, on both desktop and mobile.
Yes. The list of 11 interactive tools covers most use cases, but we study any specific request case by case: weighted quiz, multi-round voting, segmentation by participant profile, CRM or directory integration, custom forms, exports to your internal tools… Describe your need in the contact form or directly in the configurator, and we'll get back to you within 24h with a technical scope and pricing estimate.
Moderation is handled by your team via a dedicated interface accessible through a separate link. You can validate questions before publication (Q&A mode), delete messages, ban a participant, pin an announcement. In parallel, automatic moderation filters unwanted content (banned words, anti-flood, anti-spam URLs).
The tools open 15 minutes before the start of the event and automatically close 5 minutes after the broadcast. Before opening, a waiting page with a countdown is displayed for already-connected participants.
At the end of the event, you receive a complete analytics report: average watchtime, connection curve over time, engagement per tool used (Q&A, polls, quiz, idea wall…), top interaction moments. The full exchange logs are also exported as CSV. Data is kept for 90 days then automatically deleted (GDPR compliance). Video replay with synchronized tools is not included by default but can be added on request.
Les outils interactifs sont conçus comme un complément à la prestation de captation et live streaming Nomacast. Il est techniquement possible de les proposer seuls, sur une diffusion réalisée par un autre prestataire, mais cela nécessite un cadrage technique spécifique. Contactez-moi directement pour étudier ce cas.
The video broadcast has a latency of 3 to 8 seconds depending on network conditions. The tools are instant. This gap, standard on this type of broadcast, is imperceptible in the vast majority of professional event contexts. For events requiring strict real-time (sub-second), a premium solution is available on quote.
